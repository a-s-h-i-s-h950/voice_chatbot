# voice_chatbot
About the Project 

The Voice Chatbot is an interactive AI-based system that allows users to communicate using voice commands instead of typing. It integrates speech recognition, natural language processing (NLP), and text-to-speech (TTS) technologies to create a seamless conversational experience.

The chatbot listens to user input, converts speech to text, processes the query, and responds with both text and voice output—making it efficient and user-friendly.

Features
🎤 Speech-to-Text (Voice Input)
🧠 Natural Language Processing (Understanding Queries)
🔊 Text-to-Speech (Voice Output)
💬 Real-time conversation
⚡ Easy to use and lightweight
